# 🕶️ ZENLESS_CLONE // NEPS_INTERFACE

![Status](https://img.shields.io/badge/Status-Operational-yellow)
![UI](https://img.shields.io/badge/Style-Industrial_Cyberpunk-black)

Sebuah project eksperimen *Front-end* yang mengkloning estetika **Zenless Zone Zero (ZZZ)**. Fokus utama project ini adalah menciptakan antarmuka yang berat pada tipografi, kontras tinggi, dan transisi antar section yang *seamless*.

## ⚡ Core Features
- **Seamless Layout:** Desain tanpa pemisah (*zero gap*) antara Agent Roster, Contact, dan Footer.
- **Dynamic Hero:** Introduksi karakter dengan animasi *entry* yang tajam.
- **NEPS Recruitment Form:** Formulir kontak bergaya berkas kepolisian New Eridu.
- **Industrial Footer:** Penutup halaman dengan aksen *hazard stripes* dan *marquee* berjalan.

## 🛠️ Tech Stack
- **React.js** (Vite)
- **Tailwind CSS** (Custom Configuration)
- **Framer Motion** (Complex Orchestration)
- **Archivo Black Font** (Typography Focus)

## 📁 Project Structure
- `/components/Hero` - Karakter utama & slogan.
- `/components/About` - Briefing divisi.
- `/components/Character` - Roster Agent dengan auto-scroll.
- `/components/Contact` - Dossier recruitment form.
- `/components/Footer` - System status & industrial links.

## 🕹️ Getting Started
1. Clone repo: `git clone https://github.com/amirulwebdev/zenless-clone.git`
2. Install: `npm install`
3. Launch: `npm run dev`
